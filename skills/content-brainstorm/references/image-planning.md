# Image and Chart Planning Reference

## Image Types by Purpose

### 1. Custom Data Charts (Generate)
Best when: The angle's core value-add is a unique data framing that no institutional source offers.

**Chart archetypes:**
- **Scenario Tree** — Actual data path branching into probability-weighted forward paths (green/amber/red). Best for geopolitical or macro binary events.
- **Divergence Chart** — Two lines that should move together but are diverging. Best for policy traps, correlation breaks, or structural shifts.
- **Three-Panel Dashboard** — Three mini-charts side by side, each showing one signal. Best for "three converging signals" posts.
- **Cross-Asset Heatmap** — Color-coded dashboard rows (green/amber/red) with asset, level, and signal. Best for multi-asset snapshot posts.
- **Map Infographic** — Geographic map with data callouts and scenario boxes. Best for geopolitical chokepoint or regional risk posts.
- **Before/After Bar** — Grouped bars showing expectation vs. actual, or prior vs. current. Best for earnings beats, GDP misses, or consensus errors.
- **Timeline / Event Calendar** — Horizontal timeline with annotated events. Best for "what to watch this week" posts.

**Design standards:**
- Dark theme: background #0f0f23 or #1a1a2e, white text, colored data
- Font: Sans-serif (Inter, Source Sans, or similar)
- Size: 1200x627px (LinkedIn single image) or 1080x1080px (carousel/square)
- Branding: "Algo724 | Morning Intelligence" watermark, bottom-right, 8pt, alpha 0.4
- DPI: 300
- Minimum font size: 11pt for legibility
- Avoid low contrast: ALL text must be clearly visible against background

### 2. Institutional Reference Charts (Cite)
Best when: An authoritative source has already visualized the data well and their branding adds credibility.

**Common sources:**
- CNBC — Dot plots, earnings charts, market dashboards
- Reuters / LSEG — Commodity price charts, sector performance
- Bloomberg — Cross-asset snapshots, yield curves
- FRED — Historical macro data (GDP, PCE, employment)
- BullionVault — Gold vs. macro overlays
- Trading Economics — Rate paths, commodity historical
- Bondsavvy — Fed dot plot analysis

**Rules:**
- Always credit the source in the post text
- Reference images supplement, never replace, custom charts
- Check if direct download is allowed (some sources block hotlinking)

### 3. AI-Generated Images
Best when: Need a conceptual/thematic visual rather than a data chart.

**Use cases:**
- Abstract representations (e.g., "policy trap" as a visual metaphor)
- Thematic headers for series branding
- Social media thumbnails

## Image Selection Decision Tree

```
Is the post's core value a unique data framing?
├── YES → Custom Chart (match archetype to post structure)
│   ├── Binary event / forward scenarios → Scenario Tree
│   ├── Two things diverging → Divergence Chart  
│   ├── Multiple signals converging → Three-Panel Dashboard
│   ├── Cross-asset snapshot → Heatmap Dashboard
│   ├── Geographic risk → Map Infographic
│   └── Beat/miss or comparison → Before/After Bar
├── NO → Is there a high-quality institutional chart that fits?
│   ├── YES → Reference Chart (cite source)
│   └── NO → AI-generated thematic image or no image
```

## Carousel Planning (Multi-Image Posts)
For posts with complex data, consider a 2–4 slide carousel:
- Slide 1: Hook visual (the most striking single chart)
- Slide 2: Supporting data (cross-asset dashboard or probability matrix)
- Slide 3: Scenario framework or timeline
- Slide 4: Summary / CTA card with key takeaway

Carousel posts get ~1.4x more reach on LinkedIn than single-image posts.
